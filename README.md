# System Monitor Dashboard

A real-time system monitoring dashboard for tracking workstations' CPU, RAM, disk usage, and applications.

## Features

- 🔐 Multi-user authentication (arena, test1, test2)
- 📊 Real-time monitoring of CPU, RAM, and disk usage
- 💻 Track active and idle applications
- 🔄 Auto-refresh dashboard (10 seconds)
- 📱 Responsive design for mobile and desktop
- 💾 Persistent data storage with PostgreSQL (Render) or SQLite (local)

## Tech Stack

- **Backend:** Flask (Python)
- **Database:** PostgreSQL (Render) / SQLite (local)
- **Frontend:** HTML, CSS, Jinja2 templates
- **Deployment:** Render.com
